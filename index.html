Problem Statement

250M+ Indian students lack career guidance. First-gen, rural, and low-income students pick careers by chance — not design.

🤖

AI Logic / Decision Model

Claude LLM processes student profiles using a 6-step pipeline: interest mapping, goal weighting, academic filtering, geo-context, scholarship matching, and narrative generation.

📱

User Flow

3-step wizard → Arrow AI generates 4 career matches, 6 university recommendations, a 5-step roadmap, and 4 real scholarships in seconds.

🌍

Expected Social Impact

Democratizes career counseling. Replaces ₹50,000 counselors with free AI guidance for first-generation students.

⚠️

Limitations

Cannot replace a human mentor. Scholarship data needs real-time verification. University seat availability changes yearly.

🚀

Future Scope

CUET/NTA API for direct applications, WhatsApp bot for rural access, regional language voice interface, and alumni mentorship network.

Decision Model
How Arrow AI Thinks
A 6-stage AI pipeline combining rule-based logic with LLM reasoning.
01
Profile Parsing
Extracts academic background, geographic, and socioeconomic context
02
Interest Mapping
Cross-references subjects + hobbies against 50+ career-personality clusters
03
Goal Weighting
Weighs salary, passion, security, and location preferences into a scoring model
04
University Match
Filters universities by program fit, income, entrance exams, and geography
05
Scholarship Filter
Matches student profile against national, state, and private scholarship criteria
06
Roadmap + Narrative
Generates a time-phased action plan with a personalised motivational message
